Focus Timer keeps one work interval and one break interval visible without accounts, streaks, or notifications designed to pull you back.

## How to use it

Choose a session length, start the timer, and keep this tab available. Pause and reset controls remain reachable by keyboard. The timer runs locally and stores preferences only in this browser.

<Callout title="Keyboard">Use Space to start or pause when a form control is not focused.</Callout>
