Prompt Splitter turns a long block of instructions into scannable sections using headings, blank lines, and repeated structural cues.

## What it changes

The tool groups text; it does not rewrite the language or send it to a model. Review the suggested sections, rename them when useful, and copy the result.

<Callout title="Local by design">Input never leaves the page.</Callout>
