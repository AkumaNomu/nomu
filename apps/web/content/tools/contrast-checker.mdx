Contrast Checker compares foreground and background colors using the WCAG contrast formula. It reports the ratio and common AA and AAA text thresholds.

## Use the result well

A passing ratio is a baseline, not proof that text is comfortable to read. Consider weight, size, rendering, and the conditions in which the interface will be used.

<Aside>Color values stay in the browser and can be changed by typing a hex value or using the color input.</Aside>
