How the site is organized

The content pipeline and architecture choices that keep publishing fast and maintenance low.

Updated: February 15, 2026 Back to blog

Content Flow

  • Write posts as Markdown in `posts/`
  • Use frontmatter for title, date, tags, and category
  • Load and render posts dynamically on the client

Build Helpers

  • Generate `posts/index.json` for faster loading
  • Generate `feed.xml` for RSS readers
  • Cache key assets with the service worker

Customization

  • Theme switching through CSS variables
  • Simple UI mode for low-power devices
  • Optional analytics in `js/site-config.js`
