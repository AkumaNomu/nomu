# What even is torrenting?
Torrenting is a file distribution protocol called *BitTorrent*.
It’s a way to move large files across the internet without destroying a single server.

- Instead of:
**One server → Everyone downloads**
- It works like:
**Everyone → Everyone**

You download pieces from strangers.
Strangers download pieces from you. It's as simple as that.
