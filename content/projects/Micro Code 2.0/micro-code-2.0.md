#

Well,I joined micro club at usthb last October, And last week we got assigned our departments, and expectedly I got into the multimedia department. 

There was a posting on isci about needing members to work on Micro Code 2.0, so I filled the form to volunteer for motion design and I got added to the group.

We were tasked with making a teaser, pretty simple, and my job was to add 3D notifications to a phone.

I had 2 clips to work with, one fixed handheld shot, and 1 rotating shot. 
3D camera in after effects was giving horrible results, so I went with mocha AE on the fixed clip.

I replicated the iPhone notifications and added some blur and decreased brightness to focus more on it, with slight zoom in's. 

I was later asked to add sound effects, to which I agreed. When my colleague sent me the video she edited I was not satisfied with neither the pacing nor the clips, so I suggested some feedback and ended up making changes on my own. 

Afterwards, they wanted to add a pause screen at the end. I suggested adding a "next" button on a youtube interface, but I ended up making it Netflix themed. After multiple changes ( from them requesting small changes the timing, backgrounds, text, backgrounds again, etc ). I finished. You can see the finish product [Here]()
